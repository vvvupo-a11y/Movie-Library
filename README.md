# Movie Library - Приложение для управления коллекцией фильмов

## Радченко Иван


## Описание проекта

**Movie Library** — это графическое приложение на Python с использованием библиотеки `tkinter`, предназначенное для хранения и управления информацией о фильмах. Приложение позволяет добавлять, удалять, фильтровать и сохранять данные о фильмах в формате JSON.

### Основные возможности:

- **Добавление фильмов** с указанием названия, жанра, года выпуска и рейтинга
- **Автоматическая проверка** корректности ввода данных:
  - Год должен быть целым числом (от 1888 до текущего года + 5)
  - Рейтинг должен быть числом от 0 до 10
- **Отображение фильмов** в удобной таблице
- **Фильтрация** коллекции:
  - По жанру (частичное совпадение)
  - По году выпуска (точное совпадение)
- **Сохранение и загрузка** данных в/из JSON файла
- **Удаление** выбранных фильмов
- **Статистика**:
  - Общее количество фильмов
  - Средний рейтинг по всем фильмам

### Технологии:
- Python 3.6+
- Tkinter (встроенная GUI библиотека)
- JSON для хранения данных
